🚀 CÓMO USAR

🔧 MODO DESARROLLO (Mac/Windows/Linux)
   
   Opción A - Script helper:
   $ ./run-docker.sh dev
   
   Opción B - Docker compose directo:
   $ sudo docker compose up --build
   
   Opción C - Variable de entorno:
   $ DEV_MODE=true docker compose up

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🏭 MODO PRODUCCIÓN (Raspberry Pi)
   
   Opción A - Script helper:
   $ ./run-docker.sh prod
   
   Opción B - Docker compose con override:
   $ sudo docker compose -f docker-compose.yaml \
                         -f docker-compose.rpi5.yml up --build
   
   Opción C - Modificar docker-compose.yaml:
   Cambiar línea 16: DEV_MODE=false

📄 docker-compose.yaml (línea 16):
   
   environment:
     - DEV_MODE=true    # <-- Cambiar a 'false' para producción

📄 ffa-app/dockerfile (línea 26):
   
   ENV DEV_MODE=true    # <-- Cambiar a 'false' para producción
